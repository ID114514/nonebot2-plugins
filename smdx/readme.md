# 实用功能

## 添加文件
需要添加`./asset/data.json`  
json格式
```
{
    "key": "",
    "city":{

    }

}
```

## 指令
### 设置天气位置

```
setcity = on_command('设置天气城市',)
```
`设置天气城市 广州`

### 天气查询
```
# matcher
on_regex('(.*)天气|气温|多少度|几度')
```
`广州天气`  `北京温度` `天气`...



### 星期查询
```
# matcher
on_regex('(.*)是?(星期|周)几')
```
`12月20号是周几`  `12.25是周几`  `明天是星期几` ...
